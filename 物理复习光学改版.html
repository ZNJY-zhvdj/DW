🔬 光学干涉·衍射·偏振 核心考点

1. 等厚干涉（空气劈尖）

✅ 等厚含义：同一级条纹对应空气膜厚度相同。

📐 相邻明/暗纹对应的厚度差：

Δd = λ/2

📏 微小厚度/直径计算公式：

d = N · λ/2 （N 为条纹数）

2. 杨氏双缝干涉实验

🌈 单色光条纹特点：等间距、平行、明暗相间的直条纹，中央为零级明纹。

☀️ 白光照射现象：

  • 中央明纹：白色
  • 两侧：彩色条纹（紫色在内侧，红色在外侧）
  • 高级次：条纹重叠、模糊

3. 杨氏双缝干涉计算

📏 相邻明/暗纹间距：

Δx = (L · λ) / d

🔁 已知间距求入射光波长：

λ = (d · Δx) / L

说明：d 为缝间距，L 为屏到缝的距离，λ 为入射光波长。

4. 弗琅禾费单缝衍射

⚫ 暗纹条件：

a sinθ = kλ （k = ±1, ±2, …）

⚪ 明纹条件（近似）：

a sinθ = (2k+1)λ/2 （k = ±1, ±2, …）

🧩 半波带个数与明暗纹判断：

  • 暗纹：半波带数 N = 2k（偶数，相邻半波带光抵消）
  • 明纹：半波带数 N = 2k+1（奇数，剩余一个半波带的光未被抵消）

说明：a 为缝宽，θ 为衍射角。

5. 光栅方程

📊 光栅方程：

(a+b) sinθ = kλ （k = 0, ±1, ±2, …）

📈 最高衍射级次（|sinθ| ≤ 1）：

k_max = ⎣ (a+b)/λ ⎦ （取整数部分）

说明：a+b 为光栅常数。

6. 马吕斯定律

💡 线偏振光通过偏振片后的光强：

I = I₀ cos²α

说明：I₀ 为入射线偏振光强，α 为入射光振动方向与偏振片透振方向的夹角。

7. 光的偏振原理应用

  • 🌅 早晨/傍晚拍照不清晰：水面或地面反射光为水平偏振光，使用偏振镜并旋转至消光位置，可有效减弱反射光，提升画面清晰度。
  • 🪟 白天橱窗看不清内部：玻璃反射光为线偏振光，加偏振片并适当旋转，可消除反射眩光，看清内部物体。
