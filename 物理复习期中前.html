🔥 热学 + 简谐振动 核心考点总结

一、理想气体（第七章 气体动理论）

1. 理想气体状态方程

宏观式：

pV = νRT = (m/M)RT

微观式：

p = nkT

说明：n = N/V 为分子数密度，k = R/Nₐ 为玻尔兹曼常量。

2. 理想气体压强公式

p = (2/3)nε̄ₖ = (1/3)nmv̄²

微观意义：压强是大量分子频繁碰撞器壁的平均效果，与分子数密度 n 和分子平均平动动能 ε̄ₖ 成正比。

3. 理想气体温度公式

ε̄ₖ = (3/2)kT

微观本质：温度是分子平均平动动能的量度，只与温度 T 有关。

4. 自由度 & 能量均分定理

  • 单原子分子（He、Ne）：自由度 i = 3（平动）
  • 双原子分子（H₂、O₂，刚性）：i = 5（3平动+2转动）
  • 多原子分子（H₂O、CO₂）：i = 6（3平动+3转动）

能量均分定理：平衡态下，分子每个自由度的平均能量均为 (1/2)kT。

一个分子的平均总能量：

ε̄ = (i/2)kT

5. 理想气体内能

内能 = 所有分子热运动动能的总和（只与温度、物质的量、自由度有关）

E = ν · (i/2)RT

内能增量：

ΔE = ν · (i/2)R ΔT

二、热力学第一定律与典型过程（第八章 热力学基础）

1. 热力学第一定律

文字描述：系统吸收的热量 = 内能增量 + 系统对外做功

Q = ΔE + W

符号规定：

  • Q > 0 吸热；Q < 0 放热
  • W > 0 系统对外做功；W < 0 外界对系统做功
  • ΔE > 0 温度升高；ΔE < 0 温度降低

第一类永动机：不消耗能量而不断对外做功的机器 → 违反热力学第一定律，不可能制成。

2. 四个典型过程公式汇总

  • 等体过程（V = 常数）：
    做功 W = 0
    热量 Q_V = ν C_V ΔT
    内能 ΔE = ν C_V ΔT
    关系：Q_V = ΔE
  • 等压过程（p = 常数）：
    做功 W = pΔV = νR ΔT
    热量 Q_p = ν C_p ΔT
    内能 ΔE = ν C_V ΔT
  • 等温过程（T = 常数）：
    内能 ΔE = 0
    热量 Q = W = νRT ln(V₂/V₁) = νRT ln(p₁/p₂)
  • 绝热过程（Q = 0）：
    热量 Q = 0
    做功 W = -ΔE
    内能 ΔE = ν C_V ΔT

其中 C_p = C_V + R，γ = C_p / C_V。

3. 循环过程 & 热机效率

循环一周：ΔE循环 = 0，净吸热 = 净做功：Q净 = W净

热机效率：

η = W净 / Q吸 = 1 - |Q放| / Q吸

4. 卡诺循环

卡诺热机效率（T₁ 高温热源，T₂ 低温热源，热力学温标）：

ηC = 1 - T₂ / T₁

卡诺制冷系数：

εC = T₂ / (T₁ - T₂)

5. 热力学第二定律

  • 开尔文表述：不可能从单一热源吸热使之完全变成功而不产生其他影响。
  • 克劳修斯表述：热量不能自动从低温物体传到高温物体。

本质：一切与热现象有关的宏观过程都是不可逆的。

三、简谐振动（第九章 机械振动）

1. 弹簧振子测质量

角频率 ω = √(k/m)，周期 T = 2π √(m/k)

已知劲度系数 k，测周期 T 可得质量：

m = kT² / (4π²)

2. 旋转矢量法

  • 矢量长度 = 振幅 A
  • 旋转角速度 = ω
  • t = 0 时与 x 轴夹角 = 初相位 φ
  • 矢量在 x 轴上的投影：x = A cos(ωt + φ)

作用：直观判断任意时刻的位移、速度方向、相位关系。

3. 简谐振动的能量特征

总机械能守恒：

E = (1/2) k A² = (1/2) m ω² A²
  • 最大位移处（x = ±A）：动能 Eₖ = 0，势能最大
  • 平衡位置（x = 0）：势能 Eₚ = 0，动能最大

4. 同方向同频率简谐振动的合成

分振动：x₁ = A₁ cos(ωt + φ₁)，x₂ = A₂ cos(ωt + φ₂)

合振动：x = A cos(ωt + φ)

合振幅：

A = √[A₁² + A₂² + 2A₁A₂ cos(φ₂ - φ₁)]
  • 相位相同（Δφ = 2kπ）：A = A₁ + A₂（加强）
  • 相位相反（Δφ = (2k+1)π）：A = |A₁ - A₂|（减弱）
